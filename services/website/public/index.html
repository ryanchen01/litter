Features
📡

auto-discovery

Finds Codex servers on your local network automatically. No manual configuration needed.

🔑

ssh connections

Connect to remote machines over SSH. Litter can start Codex on your server automatically.

🎙

voice input

Talk to Codex using real-time voice. Transcription and conversation happen seamlessly.

🎨

70+ themes

Pick from dozens of editor themes — Catppuccin, Dracula, Nord, Gruvbox, Solarized, and more. Dark and light.

🍎

native on both platforms

Built with SwiftUI on iOS and Jetpack Compose on Android. Feels right at home.

🦀

shared rust core

Transport, session management, and state live in a shared Rust library for consistency and speed.

How it works

three steps, zero config

1

open litter

The app discovers machines on your local network and via Tailscale automatically.

2

pick a server

Select a machine and litter starts Codex on it for you. SSH, local, or Tailscale — it just works.

3

start coding

Pick up exactly where you left off. All your sessions, threads, and context are right there.
